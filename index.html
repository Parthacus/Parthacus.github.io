JavaScript is a new language for me, I only started using it to translate projects from python so I could use them on this site. I've begun using it to create programs using the Genetic Algorithm as seen in my project "Darwinian Words" which uses the powers of evolution and selective breeding to get to the entered word. You can apply this concept of selective breeding to other projects such as creating an ecosystem where there is a predator and a prey and they partake in an arms race and another famous application of the Genetic Algorithm is MarI/O which learns to play Mario using selective breeding. You can also combine neural networks and the Genetic Algorithm, in this combinations weights are not decided through supervised learning and correcting weights but instead use selective breeding to get better.
